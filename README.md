# 🧠 AI-Work - Disease Prediction Web App

---

## 🧠 Project Overview

**AI-Work** is a simple AI-powered web application that predicts possible diseases based on user-entered symptoms. It uses a dataset of symptoms and applies AI logic to suggest likely health conditions.

The system is built with a **web interface (HTML, CSS, JavaScript)** and a **Python backend**, making it easy for users to interact and get quick health insights.

---

## 🎯 Objective

- Predict diseases based on symptoms  
- Provide quick preliminary health suggestions  
- Make healthcare awareness more accessible  
- Build a simple AI-based diagnostic tool  

---

## ⚙️ Features

- Symptom-based disease prediction  
- Interactive web interface  
- Fast response system  
- Simple and user-friendly design  

---

## 🧰 Tech Stack

- HTML  
- CSS  
- JavaScript  
- Python  
- CSV Dataset  

---

## 📊 Dataset

- `disease_symptoms.csv` contains symptom-disease mappings  
- Used for training and matching user inputs  

---

## 🚀 How It Works

User enters symptoms → JavaScript/Python processes input →  
Matches dataset → Predicts possible disease → Displays result
